== Closest Fibonacci
   This gem adds instance method closest_fibonacci to Fixnum class. 
   This results in an object of Fixnum when sent closest_fibonacci message to output 
   the maximum fibonacci number less than or equal to that object.
    <b>Note</b> if fixnum object is a non-positive integer then closest_fibonacci is designed to return nil.
    <b>Example</b> puts 2.closest_fibonacci ==> 1

== Closest Fibonacci
   This gem adds instance method 'closest_fibonacci' to Fixnum class. 
   This results in an object of Fixnum when sent closest_fibonacci message to output the maximum fibonacci number less than that object.
    Note:  'closest_fibonacci' message sent to non-positive Fixnum object returns nil.
    Example: puts 2.closest_fibonacci ==> 1
